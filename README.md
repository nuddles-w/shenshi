# 神识 (shenshi)

> **每个创作者的风格，都可以被蒸馏成一个可调用的分身。**

丢一个创作者的主页链接，神识扫描其内容全貌，炼化出内容 DNA，烙印成一个可复用的创作者分身 skill——之后随时调用，以 TA 的风格起草内容。

适用于小红书、抖音、TikTok、Twitter、微博、播客、YouTube 等任意平台。

## 效果预览

蒸馏 Trader韭（小红书财经博主），生成 `trader-jiu.skill.md`：

```
【身份锚】
华尔街交易员 × 擦边内容博主
核心公式：极端专业 × 极端娱乐 = 极致传播

【选题逻辑】
高频类型：市场点评 / 擦边段子 / 华尔街见闻 / 投资避坑
触发机制：热点事件 + 自身经历 + 反常识角度三选一
禁区：从不做纯干货教程，从不摘掉"擦边"标签

【表达 DNA】
开头：反常识 + 身份背书（"在华尔街十年，我见过最蠢的操作是..."）
语气：精英感 + 自黑，英文术语 + 中文口语混搭
结构：短句 + 数据 + 金句收尾
情绪：犀利 × 幽默，绝不煽情

【护城河】
✅ 能学：Bio 四层漏斗、命名公式、内容栏目化
❌ 不能抄：擦边路线（身份不同照搬会翻车）
🔀 怎么转化：把"擦边"换成你自己的差异化钩子
```

生成 skill 后，直接调用：
```
用 trader-jiu 的风格写一条关于美联储降息的内容
```

## 用分身改写内容

蒸馏完成后，调用生成的 skill 把平淡的原始想法改写成符合该博主风格的内容：

**原始表述：**
```
美联储最近可能要降息了，这对股市应该是个好消息。
大家可以关注一下相关机会，但也要注意风险。
```

**调用 skill：**
```
用 trader-jiu 的风格改写这段内容，发小红书
```

**改写结果：**
```
在 PIMCO 经历过三次降息周期。

每次市场欢呼"牛市来了"的时候，
我们交易台上的人都在悄悄减仓。📉

降息 ≠ 股市涨。
降息 = 美联储承认经济出问题了。

真正的机会不在 A 股，在你想不到的地方。

评论区告诉我：你觉得这次降息，哪个资产会最先反应？
```

| 原始表述 | 改写后 | 套用的技巧 |
|---|---|---|
| 美联储最近可能要降息了 | 在 PIMCO 经历过三次降息周期 | 身份锚：用经历替代陈述 |
| 这对股市应该是个好消息 | 降息 ≠ 股市涨 | 反常识开头：打破预期 |
| 大家可以关注一下相关机会 | 真正的机会不在 A 股 | 犀利语气：制造悬念 |
| 但也要注意风险 | （删除） | 护城河：废话不发 |
| （无） | 评论区告诉我：你觉得... | CTA：引战式提问 |

> 改写后的内容不是"抄"Trader韭，而是把他的表达逻辑迁移到你自己的话题上。你的身份、经历、观点仍然是你的——skill 只负责改变包装方式。完整示例见 [examples/how-to-publish.md](examples/how-to-publish.md)。

## 蒸馏了什么

不是搬运内容，是炼化六层 DNA：

| 层次 | 炼化内容 |
|---|---|
| **身份锚** | 赛道定位、差异化标签、一句话人设公式 |
| **选题逻辑** | 高频话题类型、触发机制、禁区 |
| **表达 DNA** | 开头模式、语言风格、结构偏好、情绪基调 |
| **互动策略** | CTA 套路、评论区运营、粉丝关系定位 |
| **平台适配** | 算法利用方式、发布节奏、跨平台改造 |
| **护城河** | 什么能学、什么抄不了、怎么转化成自己的 |

## 工作原理

```
创作者链接 / 内容进来
        ↓
  阶段一：感知（平台差异化采集）
        ↓
  阶段二：炼化（提炼六层 DNA）
        ↓
  阶段三：烙印（生成 creator.skill.md）
        ↓
  后续调用：以 TA 的风格创作内容
```

### 平台采集策略

| 平台 | 策略 |
|---|---|
| 小红书 | 浏览器自动化；遇登录墙提示用户 |
| 抖音 / TikTok | 标题、简介、评论区高频词；请用户提供字幕 |
| Twitter / X | Timeline、高互动推文、thread 结构 |
| 微博 | 正文、话题标签、互动数据 |
| 播客 | 请用户提供转录文本；分析节目简介和嘉宾选择 |
| YouTube | 标题、简介、字幕（如有）、评论区 |
| 通用兜底 | 请用户粘贴 3-5 条代表性内容 |

## 与女娲的关系

[女娲.skill](https://github.com/alchaincyf/nuwa-skill) 蒸馏的是**思想家的认知操作系统**（怎么想、怎么判断）。

神识蒸馏的是**内容创作者的创作操作系统**（怎么选题、怎么表达、怎么火）。

同一个「蒸馏」哲学，不同维度：

| | 女娲 | 神识 |
|---|---|---|
| 蒸馏对象 | 思想家 | 内容创作者 |
| 提取的是 | 思维框架、决策启发式 | 内容风格、表达 DNA |
| 输出 | 思维分身 skill | 创作分身 skill |
| 后续用途 | 用 TA 的思维解决问题 | 用 TA 的风格发布内容 |

## 局限

- **蒸馏不了直觉** — 框架能提取，灵感不能
- **蒸馏不了私域** — 只看公开内容，直播 / 社群策略无法完整还原
- **快照而非预言** — 蒸馏的是截止当下的策略，不代表未来方向
- **公开 ≠ 真实** — 只能基于公开信息蒸馏
