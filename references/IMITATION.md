# Imitation Generator

Use this after completing a blogger analysis, when the user wants to create
similar content with their own background.

## Template

### Name Options (generate 3)
| Option | Breakdown | Memory Point |
|---|---|---|
| | | |

### Bio (4 layers)
```
💼 [Identity anchor with company/title + unique angle]
📈 [Specific result with benchmark]
🎯 [Differentiation hook — what ONLY you can say]
🔗 [CTA — optional]
```

| Dimension | Target blogger | You | Your angle |
|---|---|---|---|
| Identity | | | |
| Credibility source | | | |
| Style | | | |
| Core hook | | | |

### Content Pillars (3-4)
| Pillar | Purpose | Format | Frequency |
|---|---|---|---|
| | | | |

### First Post Draft
Use the source blogger's tone and structure, but with the user's expertise.

Title format: [Identity signal] + [Counter-intuitive claim] + [Number] + [emoji]

Body format:
- Hook: personal struggle or counter-fact (1-2 lines)
- Credibility: one-line qualification
- Method: 3-point numbered framework
- Result: specific number + benchmark
- CTA: low-friction ask
