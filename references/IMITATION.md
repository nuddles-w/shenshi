# Imitation Template

Use this template to generate a personalized RED strategy after analyzing a target blogger.

## Template Structure

### Name Options (generate 3)
Format: `Name` | Breakdown | Memory Point

### Bio (4 layers)
```
💼/💻 [Identity anchor with company/title + unique angle]
📈 [Specific result with benchmark — use real numbers]
🎯 [Differentiation hook — what ONLY you can say]
🔗 [CTA — optional, add when you have a product]
```

## Example: Dev韭 (imitation of Trader韭)

**Source blogger**: Trader韭 — Wall Street trader + Columbia + athlete + 擦边

### Name Options
| Option | Breakdown | Memory Point |
|---|---|---|
| Dev韭 | Dev = developer identity / 韭 = self-deprecating investor slang | Clean, bilingual conflict |
| 写码养基 | 写码(coding) + 养基(fund nurturing), double meaning | Visual, warm |
| 17%的码农 | Annual return in name | Strong number hook, slightly boastful |

### Bio
```
💻 字节/阿里 资深研发 | 用工程思维做投资
📈 实盘年化 17%（5年跑赢沪深300）
☕️ 不盯盘不画线，代码写策略，下班自动跑
```

### Content Pillars
| Pillar | Purpose | Format | Frequency |
|---|---|---|---|
| 下班自动跑 | Trust — prove the system works | Signal/code screenshots | Weekly |
| 调仓日记 | Trust — real P&L builds credibility | Trade log + reflection | Per trade |
| 大厂思维炒A股 | Traffic — cross-niche virality | Framework analogy (OKR, A/B test) | 2x/month |

### Sample Post
**Title**: 大厂 P7，不盯盘不画线，年化 17% 的懒人投资法 🤖

上班写 Java，下班跑策略。

入市 5 年，前 3 年亏了 30%，后 2 年靠几行 Python 翻盘。

我不盯盘，不画线，不研究 K 线——日常工作就已经够烧脑了。

我的方法很简单：
1️⃣ 量化思维建因子模型（估值、动量、质量）
2️⃣ 每周跑一次回测，信号触发就调仓
3️⃣ 其他时间该干嘛干嘛

去年年化 17%，跑赢沪深 300 约 20 个点。

不是什么神操作，就是把工程里「数据驱动决策」搬到了股市。
