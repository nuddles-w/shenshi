# 示例：用创作者分身改写内容

> 本示例展示如何调用蒸馏后生成的 `trader-jiu.skill.md`，
> 把一段平淡的原始表述改写成符合该博主风格的发布内容。

---

## 场景

你想发一条关于「美联储降息」的内容，但原始想法很平：

**原始表述：**
```
美联储最近可能要降息了，这对股市应该是个好消息。
大家可以关注一下相关机会，但也要注意风险。
```

---

## 调用 skill

```
用 trader-jiu 的风格改写这段内容，发小红书：

美联储最近可能要降息了，这对股市应该是个好消息。
大家可以关注一下相关机会，但也要注意风险。
```

---

## skill 的改写过程

**第一步：套用身份锚**
原文没有身份背书。Trader韭 的开头必须先建立"我见过"的权威感。

**第二步：套用表达 DNA**
- 原文是陈述句，缺乏张力 → 改成反常识开头
- "应该是个好消息"太模糊 → 换成具体的华尔街视角
- "注意风险"是废话 → 换成犀利的反向提醒
- 句子太长 → 拆成短句，加 emoji 作视觉断点

**第三步：套用互动策略**
原文没有 CTA → 结尾加引战钩子，触发评论区讨论

---

## 改写结果

```
在 PIMCO 经历过三次降息周期。

每次市场欢呼"牛市来了"的时候，
我们交易台上的人都在悄悄减仓。📉

降息 ≠ 股市涨。
降息 = 美联储承认经济出问题了。

真正的机会不在 A 股，在你想不到的地方。

评论区告诉我：你觉得这次降息，
哪个资产会最先反应？
```

## 技巧对照

| 原始表述 | 改写后 | 套用的技巧 |
|---|---|---|
| 美联储最近可能要降息了 | 在 PIMCO 经历过三次降息周期 | 身份锚：用经历替代陈述 |
| 这对股市应该是个好消息 | 降息 ≠ 股市涨 | 反常识开头：打破预期 |
| 大家可以关注一下相关机会 | 真正的机会不在 A 股 | 犀利语气：制造悬念 |
| 但也要注意风险 | （删除） | 护城河：废话不发 |
| （无） | 评论区告诉我：你觉得... | CTA：引战式提问 |

---

## 说明

改写后的内容不是"抄"Trader韭，而是把他的**表达逻辑**迁移到你自己的话题上。
你的身份、经历、观点仍然是你的——skill 只负责改变**包装方式**。

护城河提示：「在 PIMCO 经历过三次降息周期」这句话，
没有真实华尔街背景的人不要照搬，会翻车。
换成你自己真实的经历背书即可。
